1. Зачем нужен package main?
Package main нужен для того, чтобы определить главный файл,
где будет главная функция main с которой начнется выполнение кода
2. Где начинается выполнение программы?
в func main()
3. Чем var отличается от :=?
Var мы используем для четкого присвоения типа данных
4. Почему при делении int на int можно потерять дробную часть?
потому что это целочисленный тип данных
5. Чем for со счетчиком отличается от for { }?
for без условий - это бесконечный цикл, а for со счетчиком имеет явный конец
6. Зачем выносить код в функции?
для того чтобы не повторять один и тот же кусок кода несколько раз
